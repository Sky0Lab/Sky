# Security Policy

SkyAI is a private, proprietary AI system currently in early development.
There is no public binary or open distribution channel at this time.

## Supported Versions

SkyAI is not yet publicly released.  
No public versions are currently supported for security updates.

When public releases become available, this section will list:
- supported versions
- end-of-support dates
- security update policies

Example (for future use):

| Version | Supported          |
| ------: | ------------------ |
| 1.0.x   | ✅                  |
| < 1.0   | ❌                  |

## Reporting a Vulnerability

If you believe you have found a security issue related to SkyAI or any
associated component, please **do not** open a public issue or discussion.

Instead, contact the maintainer directly. 

Please include, when possible:
- a clear description of the issue
- steps to reproduce
- potential impact
- any logs or context that might help investigate

We aim to:
- acknowledge receipt of a valid report within **72 hours**
- provide an initial assessment (impact / severity) within **7 days**
- communicate mitigation or fix plans where applicable

Until SkyAI is publicly distributed, security reports will be handled on a
best‑effort basis and may not result in public advisories.
